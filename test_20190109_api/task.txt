API должно оперировать с двумя коллекциями(таблицами) в базе-данных: Users и Groups. Результат выполнения должен возвращаться в виде JSON, например:
{status: "OK", id: 111, age: 32, login: "..."}

Методы API, которые нужно реализовать:

*user.getAll

user.create
Параметры: login:, pass, age
Пример результата выполнения: {"id": 111}

user.delete
Параметр: id
Результат выполнения: {"ok": true}

user.findLogin // нахождение id по логину
Параметр: login
Пример результата выполнения: {"id": 111}

user.get
Параметр: id
Пример результата выполнения: {"id": 111, "login": "Dima", "pass": "765878ac6ac766575ac...", "age": 32}

group.create
Параметр: title
Пример результата выполнения: {"id": 111}

group.delete
Параметры: id
Результат выполнения: {"ok": true}

group.addUser
Параметры: id, userId
Результат выполнения: {"ok": true}

group.getUsers
Параметры: id
Пример результата выполнения: [{id: 1, login: "...", ..}, {id: 2, login: "....", ..}]

group.deleteUser
Параметры: id, userId
Результат выполнения: {"ok": true}

group.plus // добавить в первую группу юзеров второй группы
Параметры: id1, id2
Результат выполнения: {"ok": true}

group.minus // удалить из первой группы юзеров второй группы
Параметры: id1, id2
Результат выполнения: {"ok": true}
