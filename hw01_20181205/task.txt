Задача написать программу ( можно в нашем hello ) , которая :
Имеет функции которые конвертируют базовые типы в GO (кроме complex и uintptr ) друг в друга .
Можно в нашем Hello, Можно отдельным пакетом например convert.
Простой вариант функции конвертируют попарно из одного типа во второй. ( например convert.Atoi из строки в инт )
Сложный вариант все типы в один. ( convert.String("любой тип") )
Функции создаются вне функции main() , а вызываются в ней.

Базовые типы Go следующие
bool
string
int  int8  int16  int32  int64
uint uint8 uint16 uint32 uint64 uintptr
byte // псевдоним для uint8
rune // псевдоним для int32
     // представляет Unicode код
float32 float64
complex64 complex128
